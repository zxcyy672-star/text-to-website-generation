
# Text-to-Website Generation using CodeT5+

This repository provides supplementary materials for the paper:

"An AI-Driven Framework for Generating High-Quality Websites from Textual Descriptions"

## Overview

The project investigates automatic website generation from natural language descriptions using the Salesforce CodeT5+ (codet5p-220m) transformer model.

The framework is trained to generate complete website implementations including HTML, CSS, and JavaScript directly from textual requirements.

## Repository Contents

### Architecture

- `architecture.png`
  
Illustrates the overall workflow of the proposed framework.

### Dataset Samples

- `dataset_samples.txt`

Contains representative examples from the dataset used during training.

### Generated Website Examples

The folder:

`generated_websites_by_the_model`

contains several inference examples generated by the fine-tuned model.

Each example includes:

- Input textual description
- Inference script
- Generated HTML output
- Screenshots of the rendered website

## Model

Base model:

- Salesforce CodeT5+ (codet5p-220m)

Framework:

- Hugging Face Transformers
- PyTorch

## Research Purpose

The repository is provided to improve transparency and reproducibility by sharing representative dataset samples, model outputs, and inference examples associated with the research work.
